To-Do List Application
📌 Description

This is a simple To-Do List web application built using HTML, CSS, and JavaScript.
It allows users to manage tasks by moving them between Active and Completed states using clear visual indicators.

⚙️ Features
➕ Task Control
Add new tasks
Delete tasks from the list
🔄 Task Status
🔴 Red X button = mark task as Active / Not completed
🟢 Green check button = mark task as Completed
📂 Task Views
All button → shows all tasks (Active + Completed)
Active button → shows only active tasks
Completed button → shows only completed tasks
🧹 Clear Options
Clear All → deletes all tasks
Clear Completed → deletes only completed tasks
🛠️ Technologies Used
HTML
CSS
JavaScript (Vanilla JS)
How It Works
Each task starts as Active (🔴 X state)
Clicking the green check (🟢) moves task to Completed
Filtering buttons change what tasks are shown
Clear buttons remove tasks based on condition
